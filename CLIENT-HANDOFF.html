üîë Getting Live API Keys

To accept real payments, you need to get your live Stripe API keys and send them to your developer.

Step 1: Activate Your Stripe Account

  1. Go to dashboard.stripe.com
  2. Click "Activate your account" or complete the onboarding
  3. Fill in your business details, bank account, and tax information
  4. Wait for Stripe to verify your account (usually instant)

Step 2: Get Your Live API Keys

  1. In Stripe Dashboard, make sure "Test mode" toggle is OFF (top right)
  2. Go to Developers ‚Üí API keys
  3. You'll see two keys:
    • Publishable key - starts with pk_live_
    • Secret key - starts with sk_live_ (click "Reveal" to see it)
  4. Copy both keys

Step 3: Send Keys to Developer

Security Warning

Never send API keys via regular email or text. Use one of these secure methods:

  • Option 1: Use onetimesecret.com - paste keys, get a link that expires after one view
  • Option 2: Use a password manager's secure sharing feature
  • Option 3: Share via encrypted messaging (Signal, WhatsApp)

Send these two keys to your developer:

Publishable Key pk_live_xxxxxxxxxxxxx
Secret Key sk_live_xxxxxxxxxxxxx
What happens next?

Once your developer receives the keys, they'll update the website and it will start accepting real payments within minutes.
